# Smallink
[English](README.md) | Español | [Português brasileiro](LEIAME.md)

## Descripción
Acortador de URL implementado en Ruby on Rails 8 de la forma más sencilla posible.

## Compilación y ejecución de la aplicación
1. Instalar [ruby](https://www.ruby-lang.org/pt/downloads/)
2. Configurar [Ruby on Rails](https://guides.rubyonrails.org/install_ruby_on_rails.html)
3. Mueva el cmd a la carpeta del proyecto y ejecute el comando para instalar todas las dependencias: <br/>
`bundle install`
4. Ejecute el servidor de aplicaciones: <br/>
`rails server`

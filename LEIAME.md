# Smallink
[English](README.md) | [Español](LEEME.md) | Português brasileiro

## Descrição
Encurtador de urls implementado em Ruby on Rails 8 o mais simples possível.

## Build e execução da aplicação
1. Instalar [ruby](https://www.ruby-lang.org/pt/downloads/)
2. Configurar [Ruby on Rails](https://guides.rubyonrails.org/install_ruby_on_rails.html)
3. Mover o prompt de comando na pasta do projeto e executar o comando para instalar todas as dependências:<br/>
   `bundle install`
4. Executar o servidor da aplicacao:<br/>
   `rails server`
